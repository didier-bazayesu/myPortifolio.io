# Didier Bazayesu - Portfolio

A modern, professional portfolio website built with React, TypeScript, and Tailwind CSS.

## 🚀 Live Site

Visit: [https://my-portifolio-io.vercel.app/](https://my-portifolio-io.vercel.app/)

> **Note:** If the site is not loading, check GitHub Actions workflow status and ensure GitHub Pages source is set to "GitHub Actions" in repository settings.

## 🛠️ Tech Stack

- **React 19** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## 📦 Installation

```bash
npm install
```

## 🏃 Development

```bash
npm run dev
```

## 🏗️ Build

```bash
npm run build
```

## 📤 Deployment

This portfolio is automatically deployed to GitHub Pages using GitHub Actions.

### Manual Deployment (if needed)

```bash
npm run deploy
```
